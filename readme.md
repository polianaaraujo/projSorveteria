Projeto de site para vendas online de uma sorveteria fictícia desenvolvido com html e css com foco em layout responsivo, organização visual e experiência do usuário.

O objetivo do projeto é praticar conceitos fundamentais de desenvolvimento web, incluindo:
- Estruturação semântica com HTML5
- Estilização com CSS3
- Organização de cards de produtos
- Hierarquia visual e design responsivo
- Navegação por âncoras

As tecnologias utilizadas foram HTML5 CSS3 e Google fonts.

Funcionalidades:
- Cabeçalho com navegação por seções
- Seção de destaques (Mais pedidos)
- Cards de produtos com imagem, descrição e preço
- Botões de ação (Comprar)
- Layout organizado por categorias
